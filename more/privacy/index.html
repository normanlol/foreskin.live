foreskin.live

privacy

The privacy of our users are extremely important to us and Foreskin.live believes their users should know how their data is collected, used, stored, and why we might store it. Below is exactly how we do it.

Please note Bitwarden's privacy notice, if you use that service. The below information excludes Bitwarden's use of data.

Please also note that the information below does not reflect the use of included services of some services, such as the link shortener's usage of hCaptcha and any other additional service that is not mentioned here.

data we store

We collect the following details without user input (besides connection):

  • IP Address (Minecraft server only)

We collect the following details with the input of users:

  • Any user uploaded content (link shortener, Minecraft server, image server)
  • User authentication keys (image server only)
  • Temporary bypass keys to prevent multiple Captchas being prevented (link shortener only, lasts 6 hours)


how we use the data

We use the data specified above for the following reasons.

  • To provide the function of our services or to enhance the experience of users (all services)
  • To prevent server abuse (Minecraft server, image server)
  • To troubleshoot errors (all servers)


how we store the user data

We store the user data mentioned above in server logs and/or the server's database. You can request this data be removed at any time by contacting the admin.
