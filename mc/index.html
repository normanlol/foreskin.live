minecraft server

We have a Java/Bedrock Minecraft server open to the public, with hosting provided by Electrohaxz.tk.

It's open to anyone as long as you follow the global community rules.

For Java, the server is compatible with any version from 1.7 to 1.16.

For Bedrock versions, any mobile version after the most recent 100 versions on the Apple app store or Google Play Store should work.

Currently, console support for the server is unavailable, but, if there is demand you can contact me to add it.
